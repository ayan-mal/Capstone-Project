# Selenium TestNG Automation Framework

## Project Overview

This project automates the GUI Elements page of the Test Automation Practice website using Selenium WebDriver, Java, TestNG, and Page Object Model (POM).

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Features

- Form Automation
- Radio Buttons
- Checkboxes
- Dropdowns
- Date Pickers
- File Upload
- Alert Handling
- Window Handling
- Shadow DOM
- Drag and Drop
- Mouse Hover
- Double Click
- Screenshot Capture
